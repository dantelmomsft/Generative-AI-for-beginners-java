# बेसिक कॅल्क्युलेटर MCP सेवा

>**Note**: या अध्यायामध्ये [**Tutorial**](./TUTORIAL.md) समाविष्ट आहे, जो तुम्हाला नमुन्यांद्वारे मार्गदर्शन करतो.

**Model Context Protocol (MCP)** सह तुमचा पहिला प्रत्यक्ष अनुभव घेण्यासाठी स्वागत आहे! मागील अध्यायांमध्ये तुम्ही जनरेटिव्ह AI मूलभूत गोष्टी शिकल्या आणि तुमचे विकास वातावरण सेट केले. आता काहीतरी व्यावहारिक तयार करण्याची वेळ आली आहे.

ही कॅल्क्युलेटर सेवा दाखवते की AI मॉडेल्स MCP वापरून बाह्य साधनांशी सुरक्षितपणे कसे संवाद साधू शकतात. AI मॉडेलच्या कधीकधी अविश्वसनीय गणित क्षमतेवर अवलंबून राहण्याऐवजी, आम्ही एक मजबूत प्रणाली तयार कशी करायची ते दाखवू जिथे AI अचूक गणनांसाठी विशेष सेवा कॉल करू शकते.

## तुम्ही काय शिकाल

या उदाहरणाद्वारे काम करताना तुम्हाला समजेल:
- Spring Boot वापरून MCP-सुसंगत सेवा कशी तयार करायची
- थेट प्रोटोकॉल संवाद आणि AI-सक्षम संवाद यामधील फरक
- AI मॉडेल्स कधी आणि कसे बाह्य साधने वापरायचे ते कसे ठरवतात
- टूल-सक्षम AI अनुप्रयोग तयार करण्यासाठी सर्वोत्तम पद्धती

MCP संकल्पना शिकणाऱ्या आणि त्यांचे पहिले AI टूल इंटिग्रेशन तयार करण्यास तयार असलेल्या नवशिक्यांसाठी उत्तम!

## पूर्वतयारी

- Java 21+
- Maven 3.6+
- **GitHub Token**: AI-सक्षम क्लायंटसाठी आवश्यक. जर तुम्ही हे सेट केले नसेल, तर [Chapter 2: Setting up your development environment](../../../02-SetupDevEnvironment/README.md) येथे सूचना पहा.

## महत्त्वाचे संकल्पना

**Model Context Protocol (MCP)** ही AI अनुप्रयोगांना बाह्य साधनांशी सुरक्षितपणे कनेक्ट होण्यासाठी एक मानक पद्धत आहे. याला "पूल" समजून घ्या, जो AI मॉडेल्सना आमच्या कॅल्क्युलेटरसारख्या बाह्य सेवांचा वापर करण्यास अनुमती देतो. AI मॉडेल स्वतः गणित करण्याचा प्रयत्न करण्याऐवजी (जे अविश्वसनीय असू शकते), ते अचूक परिणाम मिळवण्यासाठी आमच्या कॅल्क्युलेटर सेवेला कॉल करू शकते. MCP सुनिश्चित करते की हा संवाद सुरक्षित आणि सातत्यपूर्ण होतो.

**Server-Sent Events (SSE)** हे सर्व्हर आणि क्लायंट्समधील रिअल-टाइम संवाद सक्षम करतात. पारंपरिक HTTP विनंत्यांप्रमाणे जिथे तुम्ही विचारता आणि प्रतिसादाची वाट पाहता, SSE सर्व्हरला सतत क्लायंटला अपडेट्स पाठवण्याची परवानगी देते. हे AI अनुप्रयोगांसाठी योग्य आहे जिथे प्रतिसाद प्रवाहित होऊ शकतो किंवा प्रक्रिया करण्यास वेळ लागू शकतो.

**AI Tools & Function Calling** AI मॉडेल्सना वापरकर्त्याच्या विनंतीनुसार स्वयंचलितपणे बाह्य फंक्शन्स (जसे की कॅल्क्युलेटर ऑपरेशन्स) निवडण्याची आणि वापरण्याची परवानगी देते. जेव्हा तुम्ही विचारता "15 + 27 किती आहे?", तेव्हा AI मॉडेलला समजते की तुम्हाला बेरीज हवी आहे, स्वयंचलितपणे आमच्या `add` टूलला योग्य पॅरामीटर्ससह (15, 27) कॉल करते आणि नैसर्गिक भाषेत परिणाम परत करते. AI एक बुद्धिमान समन्वयक म्हणून कार्य करते जो प्रत्येक टूल कधी आणि कसे वापरायचे ते जाणतो.

## जलद सुरुवात

### 1. कॅल्क्युलेटर अॅप्लिकेशन डिरेक्टरीमध्ये जा
```bash
cd Generative-AI-for-beginners-java/04-PracticalSamples/mcp/calculator
```

### 2. बिल्ड आणि रन करा
```bash
mvn clean install -DskipTests
java -jar target/calculator-server-0.0.1-SNAPSHOT.jar
```

### 2. क्लायंट्ससह चाचणी करा
- **SDKClient**: थेट MCP प्रोटोकॉल संवाद
- **LangChain4jClient**: AI-सक्षम नैसर्गिक भाषा संवाद (GitHub टोकन आवश्यक)

## उपलब्ध कॅल्क्युलेटर ऑपरेशन्स

- `add(a, b)`, `subtract(a, b)`, `multiply(a, b)`, `divide(a, b)`
- `power(base, exponent)`, `squareRoot(number)`, `absolute(number)`
- `modulus(a, b)`, `help()`

## टेस्ट क्लायंट्स

### 1. डायरेक्ट MCP क्लायंट (SDKClient)
कच्चा MCP प्रोटोकॉल संवाद चाचणी. चालवा:
```bash
mvn test-compile exec:java -Dexec.mainClass="com.microsoft.mcp.sample.client.SDKClient" -Dexec.classpathScope=test
```

### 2. AI-सक्षम क्लायंट (LangChain4jClient)
GitHub मॉडेल्ससह नैसर्गिक भाषा संवाद दाखवतो. GitHub टोकन आवश्यक (पहा [पूर्वतयारी](../../../../../04-PracticalSamples/mcp/calculator)).

**चालवा:**
```bash
mvn test-compile exec:java -Dexec.mainClass="com.microsoft.mcp.sample.client.LangChain4jClient" -Dexec.classpathScope=test
```

## MCP निरीक्षक (वेब UI)

MCP निरीक्षक तुमच्या MCP सेवेला कोड न लिहिता चाचणी करण्यासाठी एक दृश्य वेब इंटरफेस प्रदान करतो. MCP कसे कार्य करते हे समजून घेण्यासाठी नवशिक्यांसाठी उत्तम!

### स्टेप-बाय-स्टेप सूचना:

1. **कॅल्क्युलेटर सर्व्हर सुरू करा** (जर चालू नसेल):
   ```bash
   java -jar target/calculator-server-0.0.1-SNAPSHOT.jar
   ```

2. **नवीन टर्मिनलमध्ये MCP निरीक्षक इंस्टॉल आणि चालवा**:
   ```bash
   npx @modelcontextprotocol/inspector
   ```

3. **वेब इंटरफेस उघडा**:
   - "Inspector running at http://localhost:6274" सारखा संदेश शोधा
   - तुमच्या वेब ब्राउझरमध्ये तो URL उघडा

4. **तुमच्या कॅल्क्युलेटर सेवेशी कनेक्ट करा**:
   - वेब इंटरफेसमध्ये, ट्रान्सपोर्ट प्रकार "SSE" सेट करा
   - URL सेट करा: `http://localhost:8080/sse`
   - "Connect" बटणावर क्लिक करा

5. **उपलब्ध टूल्स एक्सप्लोर करा**:
   - "List Tools" वर क्लिक करा आणि सर्व कॅल्क्युलेटर ऑपरेशन्स पहा
   - तुम्हाला `add`, `subtract`, `multiply` इत्यादी फंक्शन्स दिसतील

6. **कॅल्क्युलेटर ऑपरेशन चाचणी करा**:
   - टूल निवडा (उदा., "add")
   - पॅरामीटर्स प्रविष्ट करा (उदा., `a: 15`, `b: 27`)
   - "Run Tool" वर क्लिक करा
   - तुमच्या MCP सेवेद्वारे परत आलेला परिणाम पहा!

ही दृश्य पद्धत तुम्हाला MCP संवाद कसे कार्य करते हे समजून घेण्यास मदत करते, तुमचे स्वतःचे क्लायंट तयार करण्यापूर्वी.
