## သင်လေ့လာမည့်အရာများ
ဤအပိုင်းတွင် Java ဖြင့် Generative AI ဖွံ့ဖြိုးတိုးတက်မှု ပုံစံများကို ပြသသည့် လက်တွေ့အသုံးချမှု သုံးခုကို ပြသပါမည်-
- Client-side နှင့် Server-side AI ကိုပေါင်းစပ်ထားသည့် Multi-modal Pet Story Generator တစ်ခု ဖန်တီးခြင်း
- Foundry Local Spring Boot Demo ဖြင့် ဒေသခံ AI မော်ဒယ်များကို ပေါင်းစည်းခြင်း
- Calculator ဥပမာဖြင့် Model Context Protocol (MCP) ဝန်ဆောင်မှု တည်ဆောက်ခြင်း

## နိဒါန်း

ဤအခန်းတွင် Java ဖြင့် Generative AI ဖွံ့ဖြိုးတိုးတက်မှု ပုံစံများကို ပြသသည့် **နမူနာပရောဂျက်များ** ကို ဖော်ပြထားသည်။ ပရောဂျက်တစ်ခုစီသည် လုံးဝအလုပ်လုပ်နိုင်ပြီး သင့်ပရောဂျက်များအတွက် အသုံးချနိုင်သော AI နည်းပညာများ၊ ဆောက်လုပ်ရေးပုံစံများနှင့် အကောင်းဆုံးလေ့လာမှုများကို ပြသထားသည်။

### Foundry Local Spring Boot Demo

**[Foundry Local Spring Boot Demo](foundrylocal/README.md)** သည် **OpenAI Java SDK** ကို အသုံးပြု၍ ဒေသခံ AI မော်ဒယ်များနှင့် ပေါင်းစည်းပုံကို ပြသထားသည်။ Foundry Local တွင် လည်ပတ်နေသော **Phi-3.5-mini** မော်ဒယ်နှင့် ချိတ်ဆက်ခြင်းကို ပြသပြီး Cloud ဝန်ဆောင်မှုများမလိုအပ်ဘဲ AI အက်ပ်များကို လည်ပတ်နိုင်စေသည်။

### Pet Story Generator

**[Pet Story Generator](petstory/README.md)** သည် ဖန်တီးမှုဆန်းသစ်သော အိမ်မွေးတိရစ္ဆာန်ပုံပြင်များကို ဖန်တီးရန် **multi-modal AI ပုံစံ** ကို ပြသထားသည့် စိတ်ဝင်စားဖွယ် Spring Boot ဝက်ဘ်အက်ပ်တစ်ခုဖြစ်သည်။ ၎င်းသည် transformer.js ကို အသုံးပြု၍ Browser-based AI အပြန်အလှန်လုပ်ဆောင်မှုများနှင့် OpenAI SDK ကို အသုံးပြု၍ Server-side လုပ်ဆောင်မှုများကို ပေါင်းစပ်ထားသည်။

### MCP Calculator Service (စတင်သူများအတွက် အဆင်ပြေသော MCP Demo)

**[MCP Calculator Service](calculator/README.md)** သည် Spring AI ကို အသုံးပြု၍ **Model Context Protocol (MCP)** ကို ပြသထားသည့် ရိုးရှင်းသော နမူနာတစ်ခုဖြစ်သည်။ ၎င်းသည် MCP အကြောင်းအရာများကို စတင်လေ့လာသူများအတွက် ရိုးရှင်းစွာ ရှင်းပြထားပြီး MCP Server တစ်ခုကို MCP Clients များနှင့် အပြန်အလှန်လုပ်ဆောင်နိုင်ရန် ဖန်တီးပုံကို ပြသထားသည်။

## လေ့လာမှု အဆင့်ဆင့်

ဤပရောဂျက်များကို အခြေခံအဆင့်မှ စတင်၍ အဆင့်မြှင့်တင်သွားရန် ဒီဇိုင်းပြုလုပ်ထားသည်-

1. **ရိုးရှင်းစွာ စတင်ပါ**: Foundry Local Spring Boot Demo ဖြင့် ဒေသခံမော်ဒယ်များနှင့် AI ပေါင်းစည်းပုံကို နားလည်ရန် စတင်ပါ
2. **အပြန်အလှန်လုပ်ဆောင်မှု ထည့်သွင်းပါ**: Pet Story Generator ကို အသုံးပြု၍ Multi-modal AI နှင့် ဝက်ဘ်အခြေပြု အပြန်အလှန်လုပ်ဆောင်မှုများကို လေ့လာပါ
3. **MCP အခြေခံများကို လေ့လာပါ**: MCP Calculator Service ကို စမ်းသပ်ပြီး Model Context Protocol အခြေခံအချက်များကို နားလည်ပါ

## အကျဉ်းချုပ်

**ဂုဏ်ယူပါတယ်!** သင်အောင်မြင်စွာ-

- **Client-side နှင့် Server-side AI လုပ်ဆောင်မှုများကို ပေါင်းစပ်ထားသည့် Multi-modal AI အတွေ့အကြုံများ** ဖန်တီးနိုင်ခဲ့သည်
- **ခေတ်မီ Java Frameworks နှင့် SDK များကို အသုံးပြု၍ ဒေသခံ AI မော်ဒယ်များကို ပေါင်းစည်းနိုင်ခဲ့သည်**
- **Tool Integration ပုံစံများကို ပြသသည့် Model Context Protocol ဝန်ဆောင်မှုများ** တည်ဆောက်နိုင်ခဲ့သည်
